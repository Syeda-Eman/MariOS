# MariOS
MariOS is a playful, web-based operating system simulation inspired by the classic world of Mario. Built entirely with vanilla HTML, CSS, and JavaScript, it features a retro desktop workspace environment equipped with draggable, layering application windows, a dynamic system clock, and customized built-in tools.

# Key Features
- Draggable Desktop Infrastructure: Native JavaScript drag-and-drop mechanics allow users to reposition workspace windows seamlessly.

- Intelligent Layer Sorting: Windows dynamically elevate to the foreground (z-index depth sorting) when clicked or opened.

- System Boundary Shield: Prevents application headers from sliding beneath the persistent top bar menu.

Built-in Applications:

- MariOS Journal: A dynamic, array-populated sidebar notebook for managing multi-entry developer logs.

- Mushroom Kingdom Explorer: An interactive map application containing location-based milestone data triggers.

- Tappable Icon Architecture: Interactive dual-click desktop icon selection states mirroring a real desktop interface.

# Tech Stack Used
HTML5: Formulates the application viewport windows, custom desktop grid structures, and foundational app frameworks.

CSS3: Powers absolute viewport layouts, backdrop blur transparency filter effects, and crisp text drop shadows.

JavaScript (ES6): Implements system time hooks, real-time boundary collision tracking, event loop state engines, and object data streaming.
